Sequentially compose two actions, passing any value produced by the first as an argument to the second.

>>= and =<< are the infix versions of bind.

m1 >> fun () -> m2 sequentially composes two actions, discarding any value produced by m1, like sequencing operators (such as the semicolon) in imperative languages.
