# Control (Unreleased)

Functional control abstractions for OCaml.

- *Homepage*: <http://odis.io/doc/control>


## Installation

iter can be installed with `opam`:

    opam install control

If you don't use `opam` consult the [`opam`](opam) file for build
instructions.
